Module: dom

Methods

<inner> children(element)

Returns an array of children of the passed in element. Will return a single element if the array contains only one child.
Parameters:
Name Type Description
element Node The element from which to retrieve children
Returns:
element - The children of element
Type
array | Node

<inner> closest(element, selector)

Finds the closest parent element of element that matches selector starting with the element this function was passed
Parameters:
Name Type Description
element Node The element from which to begin the search
selector string The CSS selector to match elements against
Returns:
The first ancestor element that matches the query
Type
Node

<inner> find(element, selector)

Finds an element by selector within a context. If child elements in element match selector, they are added to the return result
Parameters:
Name Type Description
element Node The element to search for children inside of
selector string The CSS selector to match elements against
Returns:
A single element or collection of elements that match the query
Type
array | Node

<inner> parent(element)

Returns the immediate parent of a given node
Parameters:
Name Type Description
element Node The element from which to return a parent
Returns:
element - The parent element of the element passed in
Type
Node

<inner> query(selector, context)

Finds an element by selector within the context of context (document.body if no context is supplied). This is a bare-bones analog of jQuery's `$()`
Parameters:
Name Type Description
selector string The CSS selector to match elements against
context Node The element to search inside of
Returns:
A single element or collection of elements that match the query
Type
array | Node

<inner> siblings(element)

Returns siblings of a given node. If a selector is specified, it will return only the siblings that match that selector.
Parameters:
Name Type Description
element Node The element from which to return a parent
Returns:
element - The sibling(s) of the element passed in.
Type
array | Node
